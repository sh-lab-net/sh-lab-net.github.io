Yukon (PlayingCards)

Last Updated: 07/16/2022

概容

トランプ(PlayingCards)のソリティアの1つである「Yukon」と言うゲームです。

プライバシーポリシー

このアプリケーションは、個人情報を収集したり公開しません。

This application does not collect or publish any personal information.

遊び方

ゲームの目的

全てのカードを組札に移動する。

ルール

  • 場札の先頭のカードと色違いで数字が1つ小さい任意のカードを場札の先頭に移動可能です。
  • 場札のうちカードが存在しない列には他の列のKのカードを移動可能です。
