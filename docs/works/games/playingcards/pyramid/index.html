Pyramid (PlayingCards)

Last Updated: 12/22/2019

概容

トランプ(PlayingCard)のソリティアの1つである「Pyramid」と言うゲームです。

プライバシーポリシー

このアプリケーションは、個人情報を収集したり公開しません。

This application does not collect or publish any personal information.

遊び方

ルール

ピラミッド状に配置した場札と、手札、捨て札の一番上のカードをQ(12)とA(1)、6と7等、足して13になる組み合わせで取り除いていき、場札を全て取り除くとクリアです。

K(13)は単体で取り除くことが可能です。

ジョーカー(全ての数字の代わりになるカード)が2枚使用可能です。

リディール(山札がない場合に捨て札を山札にする)は無制限です。

少ないリディール回数でのクリアを目指しましょう。
