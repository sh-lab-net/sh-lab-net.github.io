Golf (PlayingCards)

Last Updated: 08/15/2019

概容

トランプ(PlayingCard)のソリティアの1つである「Golf」と言うゲームです。

プライバシーポリシー

このアプリケーションは、個人情報を収集したり公開しません。

This application does not collect or publish any personal information.

遊び方

ルール

  • 手札のカードの数字+1もしくは-1のカードを場札の一番下から取り除いて次の手札とし、場札のカードを全て取り除けばゲームクリアです。
    • ババ(Joker) は全ての札から繋がります。
  • 場札を取り除く事ができない手札の場合、山札から一枚引いて次の手札とします。
  • 場札が残っている状態で山札が無くなるとゲームオーバーです。

ルール設定

このアプリでは次のルールを変更できます。

  • A→K, K→A と取り除くことができるか。
  • ババ(Joker)の有無
    • なし、一枚、二枚の三種類が設定可能。
