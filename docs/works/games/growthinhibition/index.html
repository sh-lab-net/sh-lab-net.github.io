GrowthInhibition

Last Updated: 11/01/2020

概容

GrowthInhibitionはライフゲームをベースにした難易度の高めのパズルゲームです。

プライバシーポリシー

このアプリケーションは、個人情報を収集したり公開しません。

This application does not collect or publish any personal information.

遊び方

ルール

  • タップでセルの生死を変更します。
  • 一定回数の変更で次の世代に進み一段上に移動します。
  • 生きてるセルが赤いラインを越えるとゲームオーバーです。
  • 死んでるセルの周囲8マスに生きてるセルがちょうど3つあると誕生します。
  • 生きてるセルの周囲8マスに生きてるセルが2または3つあると生存します。
  • 生きてるセルの周囲8マスに生きてるセルが1以下だと疎。
  • 青いセルは疎、赤いセルは密では死亡しません。
